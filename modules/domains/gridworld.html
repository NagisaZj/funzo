funzo.domains.gridworld 2D Grid world¶

The grid world is a classic Reinforcement learning (RL) domain. The world is defined by a collection of cells in a 2D grid. Cells can be:

  1. Free
  2. Blocked (obstacle)
  3. Terminal (indicating the goal position) and end of episode

The agent has to move from any start position to the goal position while avoiding blocked cells. The agent incurs travel cost to encourage shorter paths and also heavy penalties for getting into blocked cells.

Action set: (LEFT, RIGHT, UP, DOWN and STAY).

class funzo.domains.gridworld.GridWorld(gmap)[source]¶

GridWorld domain

A discrete world with cells (free, obstacles, and goal). The main task is to find a path from any start cell to a goal cell. The start, goal and obstacle cells are specified using a matrix or list of lists.
